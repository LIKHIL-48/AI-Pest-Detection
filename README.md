# AI-Powered Pest Detection and Intelligent Irrigation System

## Overview
This project uses AI (YOLOv8) for pest detection and IoT sensors for smart irrigation.

## Features
- Real-time pest detection
- Automatic irrigation control
- Water saving system

## Technologies
- Python
- YOLOv8
- OpenCV
- IoT

## How to Run
python main.py

## Results
- Accuracy: 94.8%
- Water Saving: 28%
- Response Time: 1.2 sec
